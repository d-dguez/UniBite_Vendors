# UniBite_Vendors
Repository for UniBites Vendors

UniBite Vendors is the vendor-side of **UniBite**: A mobile app which lets students and teahcers pre-order food from in campus cafeterias, 
thus cuting waiting times 

## Features
- View incoming food orders
- Manage order status
- Prototype interface 

## Tech Stack
- React Native
- JavaScript
